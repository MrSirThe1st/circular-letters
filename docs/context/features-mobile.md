# Mobile App Features

## User Screens

### Home (Sermon List)
- Display sermons newest first
- Show: title, date, preview text (first 100 chars)
- Audio indicator if available
- Tap to open sermon detail

### Sermon Detail
- Full sermon text (clean, readable)
- Audio player (if available)
  - Play/pause
  - Seek bar
  - Current time / duration
  - Background playback (expo-av)
- Continue listening when app backgrounded

### Search
- Search bar (title/date)
- Filter results real-time
- Later: search within sermon content

## Navigation
- Bottom tabs (later phase)
- Stack navigation for sermon detail

## Offline (later phase)
- Download sermon for offline reading
- Download audio for offline listening
- Cached sermon list
